# 67. Add Binary

## Example 1:

    Input: a = "11", b = "1"
    Output: "100"

## Example 2:

    Input: a = "1010", b = "1011"
    Output: "10101"

## Solution:

```javascript
/**
 * @param {string} a
 * @param {string} b
 * @return {string}
 */
var addBinary = function (a, b) {
  return (BigInt(`0b${a}`) + BigInt(`0b${b}`)).toString(2);
};
```
